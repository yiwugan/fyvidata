 环境变数
你可以用 set 指令设定一些有用的环境变数，这些变数也会随著你的人物被储
存下来，如果有不需要的环境变数，也可以用 unset 删除。详细的使用方法请
看 help set 及 help unset。
以下是目前有用的环境变数：
brief           <任何非零的值>  设定移动时只看所在地简短的名称，如果您
                                的网路速度不是很快，或者是对区域已经十
                                分熟悉，建议您打开 brief模式以减轻网路
                                负担。
brief2                          设定移动时只看所在地简短的名称和人和物。
brief3                          设定移动时只看所在地简短的名称和人和物
                                及出口。
                                
wimpy           <百分比>        当您的精气神低於这个百分比时就会自动找
                                机会逃命。
no_teach        <任何非零的值>  设定你现在不教你的徒弟技能，由於教徒弟
                                要花费一些精神，当徒弟不乖的时候....。
savemyass       <任何指令>      当您的精气神低於wimpy百分比时，这个指令
                                会由系统自动输入。
no_tell         <id/all>        使玩家（id）或所有玩家（all）都无法和你
                                说话。
revive          <1 或者 0>      如果是 1，他人可对你施心识神通，如果是0
                                则不能。缺省设置为0.
                                
sacrifice       <1 或者 0>      如果是 1，允许三清弟子牺牲你来炼丹,如果
                                是0 则不允许。缺省设置为 0.
